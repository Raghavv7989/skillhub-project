# Skill Hub (MERN Stack)

## Folder Structure
- `frontend/` - React + Vite
- `backend/` - Express + MongoDB (Mongoose)

## Setup
### Frontend
```bash
cd frontend
npm install
npm run dev
```

### Backend
```bash
cd backend
npm install
cp .env.example .env  # if you create it later
npm run dev
```

Server default: `http://localhost:5000`
